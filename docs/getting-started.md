# Getting Started

## Python Requirement

- Python 3.11 or above

## Create Virtual Environment

```bash
python -m venv .venv
source .venv/bin/activate
```

## Install

```bash
pip install -e .
```

## First Run

```bash
dryclaw -p "用一句话介绍你自己"
```

## Optional: Auth Check

```bash
dryclaw --check-auth
```
