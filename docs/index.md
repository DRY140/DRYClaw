# DRYClaw Documentation

## Project Positioning

DRYClaw is a terminal-first Python AI agent CLI focused on practical local automation:

- Tool-based task execution
- File and shell operations
- Web fetch/search support
- Schedule and daemon workflows

## Core Capabilities

- Agent loop with tool calls and iterative execution
- Local session persistence
- Configurable provider and model
- Automation-friendly CLI and schedule subcommands
